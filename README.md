# Haskell Concepts Repository

---

## Concepts Covered

- **Lists**
  - Creation, manipulation, and common operations
  - Higher-order functions (`map`, `filter`, `foldl`, `foldr`)
- **Tuples**
  - Creating and accessing tuples
  - Using tuples in functions and pattern matching
- **Dictionaries / Associative Lists**
  - Implementing key-value pairs
  - Lookup, insert, and update operations
- **Operators**
  - Arithmetic, logical, and custom operators
  - Operator precedence and associativity
- **Type Classes**
  - Instances of standard type classes (`Eq`, `Ord`, `Show`, `Num`)
  - Defining custom type class instances
- **Additional Topics**
  - Recursion and pattern matching
  - Functional composition
  - Input/output in Haskell
  - Error handling and Maybe/Either types
